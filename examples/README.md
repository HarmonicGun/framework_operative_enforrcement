# Ejemplo de departamento

> Departamento ficticio pre-configurado con 3 proyectos en diferentes fases.
> Sirve como referencia de estructura y para probar scripts.

## Proyectos

| Proyecto | Fase | Semaforo | Owner |
|---|---|---|---|
| Alfa | 8 — Produccion | Verde | Ana |
| Beta | 5 — Build | Amarillo | Luis |
| Gamma | Archivo | Rojo | Carlos |

## Como usar

1. Copia los archivos a la raiz de prueba
2. Ejecuta `python3 scripts/framework_status.py full`
3. Prueba `python3 scripts/zombie_detector.py`
4. Prueba `python3 scripts/validate_registry.py`
